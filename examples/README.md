# Examples

These examples demonstrate common patterns using the Tethrspark Core API.

## Files

- `minimal.ts`: basic module registration and prompting.
- `capabilities.ts`: provider/consumer module dependencies with `provides` and `requires`.

## Run examples

The examples are TypeScript source snippets intended for copy/paste and adaptation.
If you want to run them directly, install deps with `pnpm install`, compile with
`pnpm run build`, then execute in an environment that supports ESM TypeScript output.
